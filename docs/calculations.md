# Financial Calculations

ClearLedger provides a quick financial snapshot based on four user inputs:

- Income
- Expenses
- Debts
- Assets

The application calculates two key indicators:

- Monthly Balance
- Net Worth

---

## Monthly Balance

Monthly Balance represents the difference between income and expenses.

Formula:

Income – Expenses

Example:

Income: £3,000  
Expenses: £2,200  

Monthly Balance = £800

A positive value indicates a monthly surplus.  
A negative value indicates a monthly deficit.

---

## Net Worth

Net Worth represents the difference between total assets and total debts.

Formula:

Assets – Debts

Example:

Assets: £20,000  
Debts: £15,000  

Net Worth = £5,000

A positive value indicates that assets exceed debts.

A negative value indicates that liabilities exceed assets.

---

## Calculation Behaviour

Calculations are triggered automatically when the user enters or updates values in the input fields.

The results are displayed immediately in the Summary section of the interface.

If a field is left empty, the application treats the value as zero.
