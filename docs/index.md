## Project Overview

ClearLedger is a simple browser-based application designed to provide a quick financial snapshot.

The tool allows users to enter four values:

- Income
- Expenses
- Debts
- Assets

From these values the application calculates:

- Monthly balance
- Net worth

All data remains stored locally in the user's browser using localStorage.
