# Privacy

ClearLedger is designed as a privacy-first financial snapshot tool.

The application does not collect, transmit, or store user data outside the user's browser.

---

## Local Data Storage

User inputs are stored using the browser's **localStorage** feature.

This means:

- Data remains on the user's device
- No information is sent to external servers
- Data persists between browser sessions on the same device

Stored values include:

- Income
- Expenses
- Debts
- Assets

---

## No Data Collection

ClearLedger does not include:

- User accounts
- Analytics tracking
- Advertising
- Server-side storage

The tool operates entirely within the browser environment.

---

## Resetting Data

Users can remove all stored values by clicking the **Reset** button.

This action clears the browser's local storage for the application and resets all fields.

---

## Security Considerations

Because all calculations occur locally in the browser, ClearLedger does not transmit financial information across the internet.

Users should still exercise normal caution when entering financial information on shared or public devices.
